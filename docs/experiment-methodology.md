# Metodologia de experimentação (produto)

## Princípios

1. Hipótese clara antes de olhar resultado
2. Uma métrica primária; guardrails explícitos
3. Evidência com incerteza comunicada
4. Decisão humana documentada
5. Aprendizado reutilizável

## Ciclo

Hipótese → Desenho → Coleta agregada → Análise → Decisão → Follow-up

## O que este app não faz

- Não aloca tráfego
- Não instrumenta eventos
- Não substitui review estatístico formal em decisões de alto risco
